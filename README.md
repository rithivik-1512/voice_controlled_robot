# voice_controlled_robot
this is an webots based robot with python controller working using Voice commands using the Google API.
this contains an webots based robot controller and a python code for integrating the google api to the webots designed robot
the robot performs the basic operations such as moving in 4 directions and searches for a color if we ask for and at the same time avoids collisions with the objects around.
this is a state based code for robot with 3 states normal(normal operation),search(color searching),avoid(avoiding obstacles).
